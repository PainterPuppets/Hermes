# 快速进入开发流程
输入`git clone git@github.com:PainterPuppets/Hermes.git`将仓库clone到本地

## 后端编译

进入项目目录后输入`docker-compose up`即可打开项目后台服务器

想进入docker容器内部可输入`docker-compose run web bash`

## 前端编译

进入项目目录后输入`yarn build`即可编译前端文件
